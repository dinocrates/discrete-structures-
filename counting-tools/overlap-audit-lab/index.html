Guided Investigations

1. Duplicate Contribution

Select the shared permission region. Why is 42 + 30 too large?

ExplanationThe 12 shared users appear in both totals. Subtracting the intersection once leaves each counted once.

2. Add the Center Back

Track one all-three test through every ledger operation.

ExplanationThe net contribution is 3 - 3 + 1 = 1.

3. Count the Complement

How many of 240 tests pass both checks?

Explanation38 fail at least one check, so 240 - 38 = 202 pass both.

4. Repair, Then Multiply

Why must the union be repaired before multiplying by four environments?

ExplanationFirst count unique users; then pair each one with a later environment choice.

5. Exact vs. Guaranteed

Evenly distribute 25 jobs, then make the distribution uneven.

ExplanationThe exact maximum changes. The guaranteed bound stays 5 because N and m stay fixed.
