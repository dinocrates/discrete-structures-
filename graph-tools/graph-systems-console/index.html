Guided investigations

1. Matrix Mirror

Disable the Spawn–Courtyard route. How many matrix cells change?

Observation: Two cells change: [Spawn, Courtyard] and [Courtyard, Spawn]. An undirected edge is represented symmetrically.

2. Isolate Spawn

Disable both routes incident with Spawn. What happens to its degree and the connected-component count?

Observation: d(Spawn) becomes 0. The graph becomes disconnected and contains two components.

3. Minimal Connected Map

Can you keep all five rooms connected using only four routes?

Observation: Try a connected configuration with four active routes.
