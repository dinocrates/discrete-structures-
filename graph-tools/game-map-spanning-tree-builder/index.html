Definitions

Tree: A connected graph with no cycles.

Spanning tree: A subgraph that includes every vertex and is a tree.

Connected: A path exists between every pair of vertices.

Cycle: A path that begins and ends at the same vertex without repeating other vertices.

Forest: An acyclic graph that is not necessarily connected.
