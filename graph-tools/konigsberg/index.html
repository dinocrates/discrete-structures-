Königsberg Bridge Abstraction

In 1736, Euler asked: can you walk through Königsberg crossing each of the seven bridges exactly once? The key insight was to abstract the map into a graph. Click any land mass or bridge to see its graph counterpart.

Land masses become vertices.

Bridges become edges.

The exact size or shape of each land mass does not matter.

The exact length or material of each bridge does not matter.

Only the connection pattern matters.

⚠ This is a multigraph. More than one bridge connects the same pair of land masses — for example, two bridges connect the North Bank to Kneiphof Island. Simple graphs do not allow multiple edges between the same pair of vertices. The Königsberg graph is a multigraph.
Why does this matter for Euler paths?
Euler proved that a graph has an Euler circuit (a path crossing every edge exactly once and returning to the start) only if every vertex has even degree. The Königsberg graph has all odd-degree vertices — so no such walk exists. This insight, studied in detail in §2.4, is already visible in the degree table above.
