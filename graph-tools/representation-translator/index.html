Representation Translator

Enter a graph in any representation and the tool translates it into all the others. A graph is the same structure no matter how it is written down.

INPUT FORMAT
One edge per line: A B  or  A,B
Vertex labels are any word or letter.
Format: A: B, C  (vertex then colon then neighbors)
List every vertex, even isolated ones with no neighbors.
Optional first line: Labels: A B C D
Then the n × n matrix of 0s and 1s, one row per line.

    Drag vertices to rearrange.
