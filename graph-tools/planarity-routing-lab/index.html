Keyboard and touch help

Drag vertices and diamond bend handles with a pointer. With the keyboard, focus an object and use Arrow keys to move 5 units or Shift + Arrow for 20. Enter or Space selects it. Delete removes a selected bend handle—never a graph vertex. Escape cancels an active drag or clears the selection.
